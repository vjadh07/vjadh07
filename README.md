## 👋 Hey, I'm Viraj

I'm a CS student at Arizona State who likes building things the whole way down: the backend, the systems underneath, the UI on top, and the deploy button at the end. If it can be shipped, I want to own the path from idea to the thing people actually use. Lately that means a lot of applied AI in real product workflows. Off the clock I run a streetwear label called Still Frames, so my commits and my hoodies come from the same brain.

### What I'm building

**🧠 OS Kernel** &nbsp;·&nbsp; An x86_64 monolithic kernel in C, C++, and Assembly. Virtual memory, paging, ELF loading, a round-robin scheduler, syscalls, and multicore support. Someone has to trust the computer at the very bottom.

**👕 Still Frames** &nbsp;·&nbsp; The storefront for my streetwear label. Full-stack commerce with a catalog, Stripe checkout, and the unglamorous but critical order and inventory workflows. *(React, Node, Express, MongoDB, Stripe)*

**🎥 MyVideoJournal** &nbsp;·&nbsp; A video journaling app. FastAPI backend, PostgreSQL data model, and a React interface for saving entries and actually going back to watch them.

**📦 CoreOrder** &nbsp;·&nbsp; A containerized order-management backend. Lifecycle state, stock validation, and inventory sync on normalized schemas. *(Django, MySQL, Docker)*

### Where I've worked

**💼 Software Engineering Intern**, NexEra &nbsp;·&nbsp; Shipped 10+ REST APIs across three business domains, automated backend workflows that cut manual content handling by about 40 percent, and prototyped a few AI-assisted features.

**🎓 Undergraduate TA**, Arizona State &nbsp;·&nbsp; Helped 150+ students survive MATLAB and Arduino in office hours and project debugging, with regulars improving their assignment scores by around 20 percent.
